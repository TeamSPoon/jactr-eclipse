ACT-R and Eclipse

Intended Audience: ACT-R modelers and developers who are new to Eclipse and/or jACT-R. This does not cover ACT-R modeling. Those wishing to learn more about this complex topic should visit the ACT-R website.

Abstract : Eclipse provides a standardized IDE environment for a number of programming languages and development tools. We have decided to leverage this framework to support ACT-R modeling. Currently it only supports jACT-R, however there are plans to support the original Lisp version with this tool set as well.
