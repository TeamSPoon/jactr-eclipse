Constructing Models

ACT-R models are just like any source code; simple ones are pretty much useless. The complexity makes building or loading models a bit of a pain. There are two methods: loading an external file with a textual model description, or assembling a model programmatically. Loading from an external file is the preferred mechanism as it supports validation. Programmatic assembly, while more difficult, is useful in extension development or if you just need unheard of control.
